## What's Changed

### Added
- Customizable tap zones: set each screen edge (top / bottom / left / right) to previous page, next page, or none
- Continuous mode: option to vertically center a short page after turning
- Continuous mode: adjustable spacing between images

### Changed
- The app has been renamed to VeneraX with a new logo

### Fixed
- Seamless chapter reading: fixed tap-to-turn reloading the page after scrolling
- Seamless chapter reading: fixed tap-to-turn not crossing into the next chapter on the join page
- Fixed center-on-turn only turning a single page in continuous mode
