## What's Changed

### Added
- Windows can now switch the app icon at runtime: the choice changes the window and taskbar icon while the app is running (the pinned and Start menu icon stays the built-in one) (#134)
- Two more app-icon presets to choose from: a lettermark and an illustrated version

### Changed
- Sync-skip items are now a visual checklist grouped by category, so you no longer need to type field names by hand
- In the sync config dialog, the sync mode, backup retention count, and sync proxy options moved under a collapsible header so the save button is no longer covered

### Fixed
- Fixed pressing the close (X) button while reading occasionally quitting the whole app (#137)
- Fixed an unconfirmed upload causing old backup data to sync back and reading history to roll back (#133)
- Fixed deleting a local comic leaving stale data behind, so a newly added comic could show the previous one's info (#135, #139)
- Fixed the Android home-screen icon not switching, and made the in-app logo follow the chosen icon
- Fixed some settings pages missing translations and a password field being hidden (#129)
- Fixed "Export All" falsely reporting success when the target folder could not be written, now with a clear, actionable prompt (#130)
- Fixed the undo button on the notification bar being hard to read on dark backgrounds (#136)
