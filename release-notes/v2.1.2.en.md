## What's Changed

### Added
- Download the current chapter or the whole comic with one tap from inside the reader
- Download page now supports pause / resume / cancel all, per-task pause, resume and retry, drag-to-reorder, and estimated time remaining
- New "Wi-Fi only download" switch that auto-pauses on metered networks and resumes on Wi-Fi
- New optional "parallel downloads" setting
- A notification when all downloads finish
- A clear warning when there isn't enough disk space to download
- Unfinished downloads now resume automatically after restarting the app
- Migrate WebDAV sync settings via QR code

### Changed
- Bottom navigation selected item now uses a larger rounded-rectangle highlight
- Keywords from aggregated search are now also saved to search history
- The sync-config QR code now pops up instantly, with a refreshed interface

### Fixed
- Fixed an occasional crash on iOS at startup
- Fixed WebDAV sync sometimes hanging / becoming unresponsive on iOS
- Fixed the WebDAV auto-sync switch being reset under an empty config, and an empty config still triggering an upload
- Fixed downloads getting stuck on the whole queue when one failed or was cancelled
- Added integrity checks for downloaded images, rejecting corrupted / truncated images and retrying
- Fixed update marks occasionally being lost when cancelling a follow-update check
- Cancelled follow-update tasks now disappear from the list immediately
- Fixed the left-swipe-to-favorite and right-swipe-to-go-back gestures conflicting and triggering an accidental back
- Fixed the sidebar logo not aligning with the home search box
