## What's Changed

### Added
- New "Read Later" feature with batch add support
- Real-time image enhancement powered by GPU shaders
- Customizable home page sections (reorder and show/hide)
- Image favorites category tabs can be reordered and hidden, and now sync with backup
- Multi-select batch favoriting on the history page
- Favorite badges on multi-source aggregated search results
- Move or copy favorites to other folders from the "All" folders view
- Toggle to auto-favorite the chapter cover when favoriting an image

### Changed
- Sync data entry is now an icon button inline with the search bar
- Reworked comic details page action layout and icons
- Category tab bar is now full-width with a sliding bottom underline

### Fixed
- WebDAV backup list now shows timestamps down to the second
- Release ui.Picture resources when shader processing fails
- Favorite marks on search results now refresh in real time
- Restore WebDAV config conditionally when importing a backup
- Stop wiping follow-update marks when selecting a folder
- Determine WebDAV sync direction by numeric data version
- Fix issues with the favorite dialog on the history page

### Removed
- Removed legacy Dart Web/PWA residual code

### Build / CI
- Bump version to 2.0.9 (build 210); Android release signing remains stable for in-place upgrades

**Full Changelog**: https://github.com/Kyosee/venera/compare/v2.0.8...v2.0.9
