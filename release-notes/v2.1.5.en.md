## What's Changed

### Added
- WebDAV backups now keep the 3 most recent per platform and automatically clean up older ones

### Changed
- Smoother experience when dragging the list scrollbar

### Fixed
- Fix dropped frames when turning pages quickly and stutter when the keyboard pops up in the reader (#107)
- Fix swipe-to-favorite triggering too easily (edge back gestures and diagonal scrolling no longer set it off)
- Fix notification cards sometimes not disappearing after a background task finishes (#84)
- Fix an occasional crash when a cloud backup was applied before startup finished
- Fix settings possibly being fully reset when the process is killed by the system
- Fix key data files getting corrupted on abnormal interruption, with fault-tolerant loading when data is damaged
- Fix download resume failing outright when its state file was corrupted, instead of re-downloading
- Fix the follow-updates list not refreshing during a check, and unread update marks being lost after a data restore (#106)
- Fix timestamps in task records not shown in the local format
