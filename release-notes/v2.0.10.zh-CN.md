## 更新内容

### 新增
- 阅读背景颜色，独立于 App 主题，支持按单本漫画单独配置（跟随主题 / 白色 / 灰色 / 黑色 / 米黄 / 护眼绿）
- 夜览模式：阅读页可叠加一层可调暖色 / 纯黑 / 暗红遮罩，夜间阅读更护眼
- 支持逐章节批量标记已读 / 未读
- 本地漫画兼容 Mihon 元数据，导入时自动识别

### 变更
- 「点击翻页」与「反转点击翻页」合并为单个「翻页模式」下拉（关闭 / 点击翻页 / 反转点击翻页）
- 阅读设置页按折叠分组重新排版
- 收藏的同步与自动关联操作收进更多菜单
- 源类型解析改用随备份同步的学习型注册表，替代原有硬编码映射表

### 修复
- 修复自动翻页不跨章节的问题
- 修复进度条拖动跳转落点错误
- 重写连续阅读模式为中心键滚动，消除跨章节跳动

### 移除
- 移除内置的默认源列表 URL
- 移除 Android 外部站点 deep-link 处理

### 构建 / CI
- 版本号升至 2.0.10（build 211）；Android release 签名保持稳定，支持覆盖升级安装
