## What's Changed

### Added
- Reading background color, independent of the app theme, with per-comic override (Follow theme / White / Gray / Black / Sepia / Eye-care green)
- Night-view mode: an adjustable warm / black / dim-red overlay in the reader for easier night reading
- Batch mark a whole chapter as read or unread
- Mihon metadata compatibility for local comics, with automatic detection on import

### Changed
- Merged "Tap to turn Pages" and "Reverse tap to turn Pages" into a single "Page turn mode" selector (Off / Tap to turn Pages / Reverse tap to turn Pages)
- Reader settings page reorganized into collapsible groups
- Favorites sync and auto-associate actions moved into the overflow menu
- Source type resolution now uses a learned registry synced with backups, replacing the hardcoded source-key table

### Fixed
- Fixed auto page-turn not crossing chapter boundaries
- Fixed progress-bar seek landing on the wrong page
- Rewrote continuous reading mode with center-key scrolling to eliminate cross-chapter jitter

### Removed
- Removed the built-in default source list URL
- Removed Android external-site deep-link handlers

### Build / CI
- Bump version to 2.0.10 (build 211); Android release signing remains stable for in-place upgrades
